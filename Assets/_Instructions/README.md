Этот проект разрабатывается поэтапно, каждый этап — отдельный урок для обучения.

## Порядок этапов (кратко)

1. Структура, сцены, префабы  
2. Core (Bootstrap, GameManager, EventBus, SceneLoader)  
3. Input System  
4. Cinemachine  
5. Игрок (ООП)  
6. Оружие (наследование, полиморфизм)  
7. **Враги (Factory)**  
8. **Общий интерфейс: игрок и враги** (IDamageable, урон и атака)  
9. **ИИ врагов**  
10. **Object Pool**  
11. **Модель и анимации**  
12. **Способности врагов**  
13. **Боссы**  
14. **Архитектура и ядро геймплея**  
15. Инвентарь → 16. Лут → 17. NPC и квесты → 18. Сохранение → 19. Настройки → 20. UI  

## Важные замечания

1. **Сцены и префабы** создаются в Unity Editor, следуйте инструкциям в документе Этап 1.
2. **Порядок выполнения** важен — не пропускайте этапы.
3. **Тестирование** — проверяйте работу после каждого урока.
4. **Object Pool** внедряется после появления врагов и их спавна (и снарядов), когда есть что пулить.
5. **Урок по архитектуре** (этап 14) собирает общую картину и core loop после завершения боевого ядра (игрок, оружие, враги, общий интерфейс IDamageable, ИИ, боссы).
6. **Общий интерфейс** (этап 8) — сразу после врагов: сведение игрока и врагов к IDamageable, чтобы и те и другие могли атаковать и получать урон через одну логику.
